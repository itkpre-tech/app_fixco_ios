## 🛠️ Tech Stack

* **Frontend:** Flutter

* **Tools:** Google Play Console, Apple Developer Console

---

## ⚙️ Setup Instructions

### 1. Clone the Repository

```
git clone https://github.com/itkpre-tech/fixco-app.git
cd fixco-app
```

### 2. Install Dependencies

```
flutter clean
flutter pub get
```

### 3. Run the App

```
flutter run
```

---

## 🔐 Environment Configuration

Create a `.env` file and add required API keys and configurations.
